# 海亮C9夜谭·其三（暑课辩）
时值仲夏，蝉鸣如沸。见书院告示：暑期托管，免费开放教学资源。心下诧异，分明是补课十日，怎生换了这般雅称？

忆及《周礼》有云：“以乡三物教万民。”乃知古之教化，必明其义、定其名。今之“托管”二字，妙不可言。既可解父母无暇之困，又可避有偿补课之嫌。一如古之方士，将铅汞唤作玉液，把丹毒称为琼浆。

遂致电市教体局，言及“擦边”之虞。那厢声音温润如玉：“此乃便民之举，非补课也。”又问收费几何，答曰：“自愿托管，酌情补贴。”忽想起庄子所言“名者实之宾”，而今实未至而名已彰，岂非妙事？

越数日，心有不甘，复投省厅信访。不日得复，洋洋千言：军训之规、安全之要、托管之善，独不言补课之实。恰似那卖椟还珠的楚人，将个匣子装点得锦绣灿烂，却教人忘了内中本该有的珍珠。

尤妙者，文中将我们这些日日端坐讲堂的少年，比作“无人照看之雏鸟”。我倒要问问：既是雏鸟，何不纵之山林？既欲照看，何须八九张试卷相伴？这等照看法，与那养鸡场何异——晨昏定省，只盼多下几个蛋罢了。

近来读史，见《史记·滑稽列传》中淳于髡说齐王：“饮一斗亦醉，一石亦醉。”其妙在因势利导。今观“托管”之说，深得古人之髓——同一事也，可说可不说，可如是说可如彼说，全在立言者匠心独运。

忽忆儿时读《皇帝的新衣》，常笑众人眼拙。而今方知，那说破真相的孩童，才是最不懂事的。

末了，忽见书院新贴楹联：
左书：解民生之忧殚精竭虑
右书：育栋梁之材呕心沥血
横批：甘霖普降

我站在对联下，汗水浸透了衣衫。不知这七月流火，算不算得甘霖一种？
